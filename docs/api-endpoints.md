# API Endpoints

## HTML API

### Root

- `GET /` - loads React web app

##JSON API

###Users
- GET /api/users/:id
- POST /api/users
- PATCH /api/users


###Session
- POST /api/session
- DELETE /api/session


###Listings

- POST /api/listings

- PATCH /api/listings/:id
- DELETE /api/listings/:id


###Regions
- GET /api/regions
- GET /api/regions/:id


###Bookings

- PATCH /api/bookings/:id
- DELETE /api/bookings/:id


###Reviews
